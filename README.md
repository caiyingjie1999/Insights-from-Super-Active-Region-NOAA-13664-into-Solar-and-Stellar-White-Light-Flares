# A-Solar-White-light-Flare-Identification-Method

We improved the solar WLF identification method for obtaining more solar WLF events and their accurate light curves from two aspects: 1) imposing constraints defined by the typical temporal and spatial distribution characteristics of WLF-induced signals; 2) setting the intrinsic threshold for each pixel in the flare ribbon region according to its inherent background fluctuation rather than a fixed threshold for the whole region..

The flare_region.csv utilized in the code is derived from AIA 1600 Å observations acquired during the flare interval, whereas the delta.csv is computed from HMI 6173 Å continuum intensity images captured during a pre-flare observational period.

If you use this solar white-light flare identification method, please cite our article (Statistics of Solar White-Light Flares I: Optimization and Application of Identification Methods).
